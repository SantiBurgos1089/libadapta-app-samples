# libAdapta app samples (EN)
Visual app samples using libAdapta.

This is an unofficial project that aims to bring information about how to create windows, basic apps and other controls associated with [libAdapta](https://github.com/xapp-project/libadapta) and taking the [libadapta-examples](https://github.com/xapp-project/libadapta-examples) repository as a starting point.

All the apps in this repository are created by using a mixture on my particular experience with Python, libAdwaita (the original base of libAdapta) and GTK4 as a whole. The apps might not be perfect nor something that can be considered a "final product" or something related, but my idea is to somehow (also) contribute on how frontend applications are created using the combination of the technologies previously mentioned.

I must mention that I do have used AI to generate some code (mainly ChatGPT and Google Gemini which I have access to their free tiers), **however, I must mention that the entire code has not been generated nor has been full AI as it is,** but a complement of both AI as an initial template or a base, along with searches from Google, the documentation available for GTK and a lot of trial and error since much, if not the majority, of the code generated by the AI gives errors if I just copy and paste it.


# Muestra de aplicaciones con libAdapta (ES)
Muestra de aplicaciones visuals utilizando libAdapta.

Este es un proyecto no oficial que intenta mostrar información acerca de como crear ventanas, aplicaciones básicas y otros controles asociados con [libAdapta](https://github.com/xapp-project/libadapta) y tomando el repositorio de [libadapta-examples](https://github.com/xapp-project/libadapta-examples) como punto de partida.

Todas las aplicaciones en este repositorio han sido creadas utilizando una mezcla de mi experiencia particular con Python, libAdwaita (la base original de libAdapta) y GTK4 en general. Las aplicaciones no son perfectas, no son consideradas "producto final" ni intentan considerarse algo del estilo "yo soy la referencia perfecta", mi idea es de alguna manera contribuir (también) a como se pueden realizar aplicaciones de frontend utilizando la combinacion de las tecnologías previamente mencionadas.

Debo mencionar que si he utilizado IA para generar algo de código (principalmente ChatGPT y Google Gemini, los cuales tengo acceso por sus cuentas gratuitas al público), **sin embargo, debo mencionar que la totalidad del código expuesto no ha sido hecha con IA,** sino que he utilizado una mezcla del código generado por IA como una base, junto con búsquedas en Google, la documentación disponible de GTK y bastante prueba y error ya que mucho del código generado por IA, si no es que la mayoría del mismo, da errores en general si solamente copiara y pegara dicho código generado.
